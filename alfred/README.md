# Samplewatch Alfred Workflow

Type `sd` in Alfred, then choose an action.

Useful queries:

```text
sd status
sd start
sd stop
sd phaseplant
sd p modular
sd lp phaseplant
sd x
sd d
```

The workflow calls the installed `samplewatch` command, so run `scripts/install.sh` first.
`sd status` reports whether the detached backend is running. When samplewatch notifications are enabled, the backend sends the status popup.
